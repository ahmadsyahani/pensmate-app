## 🚀 Fitur Utama

- **Admin Dashboard**: Kelola acara, jadwal (rundown), dan pantau daftar anggota.
- **Member Dashboard**: Lihat agenda mendatang, riwayat event, serta eksplorasi.
- **Presensi Digital**: Check-in mandiri menggunakan token 6 digit untuk mencatat kehadiran.
- **PensMates (Member Directory)**: Cari dan temukan sesama mahasiswa di komunitas.
- **Rundown Interaktif**: Susunan acara ala *timeline* yang rapi dan mudah dibaca.

## 🛠️ Teknologi yang Digunakan

- **Backend**: Laravel 11 (PHP 8.2+)
- **Frontend**: React.js, Inertia.js, TailwindCSS
- **Database**: SQLite / MySQL (via Eloquent ORM)
- **Animasi & UI**: Framer Motion, Lucide React, Shadcn UI-inspired Components
